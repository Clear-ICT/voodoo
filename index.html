Reproductible odoo environment

Installing Odoo manually is long, error prone and not well documented.

Don't waste your time: use voodoo

Benefit from best practices

  • Clean and functionnal directory structure out of the box
  • Keep your configuration setup under control with git
  • Manage dependencies with buildout
  • Use nice tools for develop, test and debug

Docker

Voodoo make use of docker and docker-compose. It set up a container with the tool chain you need : Postresql, Anybox's buildout recipe, ak. And provide you few shortcuts.

Read the documentation on github/akretion/voodoo
